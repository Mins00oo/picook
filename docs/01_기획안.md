## 1. 프로젝트 개요

### 1.1 프로젝트명
**Picook**

### 1.2 한 줄 요약
냉장고 속 재료로 레시피를 추천하고, **음성 코칭으로 함께 요리**하며, **유튜브 쇼츠를 단계별 레시피로 변환**해주는 iOS 모바일 앱.

### 1.3 앱 포지셔닝
```
기본기    : 재료 기반 레시피 추천 (당연히 되는 것)
킬러 피처1: 음성 코칭 모드 ("오, 이것까지 되네?")
킬러 피처2: 쇼츠 URL → 단계별 레시피 변환 ("이거 진짜 편하다")
리텐션    : 등급 시스템 — 요리할수록 레벨업 (테일즈런너처럼)
```

### 1.4 프로젝트 배경
- 초보 요리사는 냉장고 속 재료를 잘 활용하지 못해 음식물을 버리거나 배달에 의존한다.
- 기존 레시피 앱은 "뭘 만들까?"까지만 해결하고, "어떻게 동시에 잘 만들까?"는 해결하지 않는다.
- 초보가 2개 요리를 동시에 하면 시간 관리 실패로 음식이 식거나 면이 불는 문제가 빈번하다.
- 유튜브 쇼츠에 요리 영상이 많지만 너무 빨라서 따라하기 어렵다는 불만이 많다.
- 기존 시장에 "레시피 추천", "음성 리더", "멀티 타이머", "쇼츠 변환"을 하나로 합친 앱은 없다.

### 1.5 프로젝트 목표
- 즉시 조리 가능한 레시피만 추천 (재료 매칭 TOP 10)
- 음성 코칭으로 1개 요리 안내 + 2개 요리 타임라인 통합
- 유튜브 쇼츠 URL → 단계별 레시피 변환
- 등급 시스템으로 게이미피케이션
- "기다려주는 코칭" 설계 (초보 페이스에 맞춤)
- 백오피스에서 엑셀 일괄 등록으로 효율적 데이터 관리

## 2. 핵심 기능 상세

### 2.1 음성 코칭 모드

**싱글 코칭 (1개 요리)**
- 각 단계를 음성으로 읽어줌
- 능동 단계: 사용자 확인 후에만 진행 (시간 강제 없음)
- 대기 단계: 타이머 자동 → 완료 알림 → 사용자 확인 후 진행
- 느려도 재촉 안 함

**멀티 코칭 (2개 요리)**
- 2개 레시피 → 하나의 타임라인으로 통합
- 대기 시간에 다른 요리 능동 단계 배치 → 시간 최적화
- 딜레이 시 예상 완료 시간만 조용히 업데이트

**코칭 핵심 원칙**
1. 능동은 시간 강제 안 함 — 사용자 확인 후에만
2. 대기 타이머는 정확히
3. 딜레이 시 자동 재계산 (조용히)
4. 앱이 사용자 페이스에 맞춤

**on/off**: 사용자별 설정. 기본값 입문/초급→on, 중급/고급→off

**기술 핵심**: 조리 단계를 능동(active)/대기(wait)로 분류. 백오피스에서 관리자가 지정.

**멀티 코칭 예시**
```
00:00 [찌개] 두부, 호박, 파 썰기
03:00 [찌개] 물 올리고 된장 풀기 → 10분 대기
03:30 [계란말이] 계란 풀기 (찌개 끓는 동안)
05:30 [계란말이] 프라이팬에 굽기
10:30 [계란말이] 완성
13:00 [찌개] 간 맞추기
15:00 "모든 요리 완성! 🎉"
```

### 2.2 쇼츠 URL → 단계별 레시피 변환

**사용자 니즈**: 쇼츠 요리 영상이 너무 빨라서 따라하기 어려움 → 텍스트로 단계별 정리

**기술 흐름**
```
유튜브 쇼츠 URL 붙여넣기
→ 서버: 음성 추출 (yt-dlp + ffmpeg)
→ STT 텍스트 변환 (Whisper API)
→ AI 단계별 구조화 (GPT-4o)
→ 단계별 레시피로 표시
→ (선택) 코칭 모드로 바로 시작
```

**캐싱 전략**
- 변환 결과 서버 캐싱 (같은 URL → 캐시 반환)
- AI 모델 버전 같이 저장 → 모델 업그레이드 시 캐시 버전 불일치하면 재변환

**정책**: 유튜브만 (MVP), 무료 무제한, 변환 10~30초, 실패 시 안내

### 2.3 등급 시스템 (게이미피케이션)

코칭 완료 + 완성 사진 업로드 = 1회 카운트 → 레벨/칭호 상승

| 레벨 | 칭호 | 누적 완료 수 |
|------|------|------------|
| Lv.1 | 요리 병아리 🐣 | 0~2 |
| Lv.2 | 주방 탐험가 🔍 | 3~5 |
| Lv.3 | 냉장고 파이터 💪 | 6~10 |
| Lv.4 | 집밥 장인 🍳 | 11~20 |
| Lv.5 | 요리 마스터 👨‍🍳 | 21~35 |
| Lv.6 | 셀프 셰프 ⭐ | 36~50 |
| Lv.7 | 전설의 요리사 🏆 | 51+ |

본인만 보기 (MVP). 레벨업 시 축하 애니메이션.

## 3. 타겟 사용자

| 페르소나 | 특성 | 주요 활용 |
|---------|------|----------|
| 요리 초보 (20대) | 자취 시작 | 싱글 코칭, 쇼츠 변환, 등급 |
| 바쁜 1인 가구 (30대) | 퇴근 후 빠른 요리 | 멀티 코칭 |
| 쇼츠 요리 팔로워 | 쇼츠 자주 시청 | 쇼츠 변환 + 코칭 |

**관리자**: 콘텐츠(레시피·재료 엑셀 일괄등록), 운영(사용자/통계)

## 4. 기능 범위

### Phase 1 (MVP, 3~4개월)
소셜로그인(Apple+카카오), 온보딩(2단계), 프로필, 재료선택, 추천(TOP 10), 레시피상세, **싱글코칭, 멀티코칭, 쇼츠변환, 등급시스템**, 즐겨찾기

### Phase 2 (출시 후 1~2개월)
**알레르기 필터, 조리 도구 필터**, Google 로그인, 피드백, i18n, 고품질 TTS, 등급 공개, 다크모드

### Phase 3 (출시 후 3~4개월)
푸시알림, 영양정보, 네이버 로그인, 코칭 히스토리, 틱톡/인스타 릴스

### 백오피스 Phase 1
관리자 인증, 대시보드, 레시피 관리(**엑셀 일괄등록** + active/wait), 재료 관리(**엑셀 일괄등록** + 카테고리 + 동의어), 쇼츠 캐시 관리, 피드백 관리, 통계

## 5. 경쟁 분석

기존: 레시피 추천(만개의레시피, SuperCook), 음성 리더(Voicipe), 멀티 타이머(Timer+), 쇼츠(유튜브 직접 시청) — **이 네 가지를 하나로 합친 앱은 없음**.

본 앱 차별점: 재료 기반 추천, 싱글/멀티 코칭, 기다려주는 코칭, **쇼츠→레시피 변환**, **등급 시스템**, 검증된 자체 DB, 광고 없음.
