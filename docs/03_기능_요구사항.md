## 2. 사용자 앱

### 2.1 인증

#### FR-USR-001: Apple Sign-In (P0)
Apple HIG 준수. 최초 이름/이메일 즉시 저장. 서버에서 Apple 공개키 검증 → 우리 JWT 발급.

#### FR-USR-002: 카카오 로그인 (P0)
카카오 SDK 토큰 → 서버에서 kapi.kakao.com/v2/user/me 검증 → 우리 JWT 발급.

#### FR-USR-004: 로그아웃 (P0)
확인 → 토큰 삭제 → 로그인 화면.

#### FR-USR-005: 회원 탈퇴 (P0)
30일 유예 삭제. Apple 필수.

### 2.2 온보딩 및 프로필

#### FR-USR-006: 기본 정보 입력 (P0)
**2단계**: 요리 실력 → 코칭 설정. 건너뛰기 가능.
Phase 2 추가: 알레르기 입력, 조리 도구 선택.

#### FR-USR-007: 프로필 수정 (P0)
닉네임 20자, 프로필 이미지 5MB.

#### FR-USR-008: 알레르기 관리 (P1) — Phase 2
#### FR-USR-009: 조리 도구 관리 (P1) — Phase 2

### 2.3 재료 선택

#### FR-USR-010: 카테고리별 조회 (P0)
9 카테고리. 사전 등록 목록에서만 선택.

#### FR-USR-011: 재료 검색 (P0)
실시간 자동완성, 초성 검색, 동의어 매칭.

#### FR-USR-012: 재료 선택/해제 (P0)
최소1~최대30.

### 2.4 레시피 추천

#### FR-USR-013: 추천 전 확인 (P0)
재료(삭제/추가 가능) + 필터(시간/난이도/인분).

#### FR-USR-014: 레시피 추천 매칭 (P0)
매칭률 = 보유필수재료/전체필수재료 × 100%. 시간/난이도/인분 필터. 매칭률 30%+ → **매칭률 DESC → TOP 10 고정 반환.** 부족 재료 목록 포함.
MVP 제외: 조리 도구 필터, 알레르기 제외 (Phase 2).

#### FR-USR-015: 부족 재료 표시 (P0)
100% 미만: 부족 재료명. 100%: 초록 뱃지.

### 2.5 레시피 상세

#### FR-USR-016: 상세 조회 (P0)
이미지, 기본정보, 재료(보유/미보유), 조리순서(active/wait 아이콘), 팁. 조회수 24시간 내 1회.

#### FR-USR-017: 인분 조절 (P1)

### 2.6 즐겨찾기

#### FR-USR-018: 추가/삭제 (P0)
무료 20개. 쇼츠 변환 레시피도 저장 가능.

#### FR-USR-019: 목록 조회 (P0)

### 2.7 검색

#### FR-USR-020: 레시피명 검색 (P0)
#### FR-USR-021: 검색 기록 (P1)

### 2.8 피드백

#### FR-USR-022: 레시피 피드백 (P1)
맛있음/보통/어려움 + 200자.

### 2.9 코칭 공통

#### FR-USR-023: 코칭 on/off (P0)
온보딩 기본: 입문/초급→on, 중급/고급→off.

#### FR-USR-024: 음성 설정 (P0)
속도 0.8x/1.0x/1.2x. 알림음 3종. MVP: iOS 기본 TTS.

#### FR-USR-025: 음성 명령 (P0)
"다음"/"반복". iOS SFSpeechRecognizer. 80%+. 화면 탭 항상 대체.

#### FR-USR-026: 화면 탭 전환 (P0)
능동: 탭→다음. 대기 완료 후: 탭→다음. 대기 진행 중: 무시.

#### FR-USR-027: 화면잠금 방지 + 백그라운드 (P0)

### 2.10 싱글 코칭

#### FR-USR-028: 싱글 코칭 진행 (P0)
능동: 사용자 확인 후에만 진행. 대기: 타이머 자동 → 완료 알림 → 사용자 확인.

#### FR-USR-029: 일시정지/종료 (P0)

### 2.11 멀티 코칭

#### FR-USR-030: 레시피 선택 (P0) — 최대 2개
#### FR-USR-031: 타임라인 생성 (P0) — active 미중복, wait(parallel) 중 active 배치
#### FR-USR-032: 멀티 진행 + 딜레이 (P0) — 예상 완료만 조용히 업데이트, 재촉 없음

### 2.12 쇼츠 변환

#### FR-USR-033: 쇼츠 URL 변환 (P0)
유튜브만, 무료 무제한. 서버: yt-dlp → Whisper → GPT-4o. 캐싱(url_hash + ai_model_version). 모델 업그레이드 시 재변환.

#### FR-USR-034: 변환 레시피로 코칭 시작 (P0)
#### FR-USR-035: 변환 레시피 즐겨찾기 저장 (P0)

### 2.13 등급 시스템

#### FR-USR-036: 요리 완료 + 사진 업로드 (P0)
코칭 완료 + 사진 = 카운트 +1. MVP 자동 카운트.

#### FR-USR-037: 등급 표시 (P0)
Lv.1~7. 본인만 (MVP). 레벨업 축하 애니메이션.
