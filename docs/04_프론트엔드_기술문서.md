## 3. 코칭 엔진 설계

```typescript
type StepType = 'active' | 'wait';

interface RecipeStep {
  id: number; stepNumber: number; description: string;
  stepType: StepType; durationSeconds: number; canParallel: boolean;
}

interface TimelineItem {
  id: string; recipeId: number; recipeName: string; recipeColor: string;
  step: RecipeStep; scheduledStartSeconds: number;
  status: 'pending' | 'active' | 'waiting' | 'completed';
}
```

타임라인 엔진: primary(오래 걸리는 요리)의 wait(canParallel) 구간에 secondary의 active 배치. active 미중복.

## 4. API 연동 (Spring Boot)

```typescript
// src/api/client.ts
import axios from 'axios';

const api = axios.create({
  baseURL: __DEV__ ? 'http://localhost:8080' : 'https://api.picook.com',
});

// JWT 자동 첨부 인터셉터
api.interceptors.request.use(async (config) => {
  const token = await SecureStore.getItemAsync('accessToken');
  if (token) config.headers.Authorization = `Bearer ${token}`;
  return config;
});
```
