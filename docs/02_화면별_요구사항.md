## 1. 화면 구조 개요

### 1.1 사용자 앱 (iOS)

```
[스플래시]
  └─ [온보딩] (최초 1회)
       └─ [로그인/회원가입]
            └─ [기본 정보 입력] (최초 1회, 2단계)
                 └─ [메인 탭 구조]
                      ├─ 홈 탭
                      │    ├─ 홈 화면
                      │    ├─ 재료 선택
                      │    ├─ 추천 전 확인
                      │    ├─ 추천 결과 (TOP 10)
                      │    ├─ 레시피 상세
                      │    ├─ 싱글 코칭 화면
                      │    ├─ 멀티 코칭 - 타임라인 미리보기
                      │    ├─ 멀티 코칭 화면
                      │    └─ 요리 완료 (사진 업로드 + 등급)
                      ├─ 쇼츠 변환 탭
                      │    ├─ URL 입력 화면
                      │    └─ 변환 결과 화면
                      ├─ 즐겨찾기 탭
                      └─ 마이페이지 탭
                           ├─ 프로필 (등급 표시)
                           ├─ 코칭 설정
                           ├─ 앱 설정
                           └─ 계정 삭제
```

### 1.2 백오피스 (웹)

```
[로그인]
  └─ [사이드바 + 콘텐츠]
       ├─ 대시보드
       ├─ 레시피 관리 (개별 + 엑셀 일괄 + active/wait)
       ├─ 재료 관리 (개별 + 엑셀 일괄 + 카테고리 + 동의어)
       ├─ 쇼츠 캐시 관리
       ├─ 사용자 관리
       ├─ 피드백 관리
       ├─ 통계
       └─ 관리자 계정
```

## 2. 사용자 앱 — 화면별 상세

### SCR-001: 스플래시
앱 로고 + 이름 (중앙), 최대 2초. 토큰 확인 → 유효: 홈 / 무효: 온보딩.

### SCR-002: 온보딩 (최초 1회)
3페이지 스와이프. 페이지1: 재료 추천 / 페이지2: 음성 코칭 / 페이지3: 쇼츠 변환 + [시작하기]. 하단 인디케이터 + [건너뛰기].

### SCR-003: 로그인/회원가입
[Apple로 로그인] (HIG 준수) + [카카오로 로그인]. 성공 → 신규: 기본정보 / 기존: 홈.

### SCR-004: 기본 정보 입력 (최초 1회, 2단계)

**스텝 1: 요리 실력** — 입문/초급/중급/고급 (카드형 싱글 선택, 각 설명 포함)
**스텝 2: 코칭 설정** — "음성으로 안내받을 수 있어요" 안내 + 코칭 on/off (실력 기반 기본값: 입문/초급→on) + 음성 속도 (느리게/보통/빠르게)

각 스텝 건너뛰기 가능. 마이페이지에서 수정 가능.
Phase 2 추가 예정: 알레르기 입력, 조리 도구 선택.

### SCR-005: 홈 화면
상단: 인사말 + **등급 뱃지** (Lv + 아이콘). CTA: [재료로 레시피 찾기] 큰 버튼. 최근 검색: 재료 조합 5개. 인기 레시피: 가로 스크롤 카드. 즐겨찾기 미리보기: 최근 3개.

### SCR-006: 재료 선택
검색 바 (실시간 자동완성, 초성, 동의어). 카테고리 탭: 전체/채소/과일/육류/해산물/유제품·계란/곡류·면/양념·소스/기타. 그리드 2열, 아이콘+이름. **사전 등록 목록에서만 선택** (자유입력 불가). 탭→체크, 재탭→해제. 최소1~최대30. 하단: 선택 수 + 칩 + [다음].

### SCR-007: 추천 전 확인
재료: 칩 나열 (삭제/추가 가능). 필터: 시간(15/30/60/무제한), 난이도(쉬움/보통/어려움/무제한), 인분(1/2/3~4/5+). 하단: [이 조건으로 추천받기].

### SCR-008: 추천 결과 (TOP 10)
상단: 조건 요약 + [수정] + **[일반]/[멀티 코칭] 토글**. 정렬: 매칭률순 (기본). 카드: 이미지+요리명+매칭률+시간+난이도+부족재료. **결과 최대 10개 고정 표시.** 멀티 모드 시: 카드에 체크박스, 최대 2개. 하단 미니카드+[타임라인 미리보기]. 빈 상태: "만들 수 있는 레시피 없음" + [다시 선택].

### SCR-009: 레시피 상세
헤더: 대표 이미지, 뒤로, 하트. 기본정보: 요리명, 시간, 난이도, 인분, 카테고리. 재료: 보유(체크)/미보유(빨강) 구분. 조리 순서: 번호+설명+이미지+시간+**active/wait 아이콘**. 팁. **하단 고정**: [코칭 모드로 요리 시작] (코칭 on 시) + [즐겨찾기].

### SCR-010: 싱글 코칭 화면
전체화면 진입. 상단: 요리명 + 현재/전체 단계. 중앙: 현재 단계 (큰 텍스트) + 이미지. 타이머: 대기→원형 카운트다운 / 능동→경과시간. 예상 완료 시간 (딜레이 시 조용히 업데이트). 하단: 진행 바. 능동: 음성 읽기 → 사용자 확인 후 진행. 대기: 타이머 자동 → 완료 알림 → 사용자 확인 후 진행. 음성 명령: "다음"/"반복". 화면 탭: 능동→다음, 대기중→무시. 컨트롤: 일시정지, 닫기. 화면잠금 방지. 백그라운드 타이머+푸시. 완료 → SCR-013.

### SCR-011: 멀티 코칭 — 타임라인 미리보기
2개 요리명 + 예상 총 시간. 세로 타임라인 (시간순, 요리A 파랑/B 주황, active/wait 아이콘). "예상 시간이며 페이스에 맞춰 조정됩니다". [코칭 시작] + [레시피 변경].

### SCR-012: 멀티 코칭 화면
현재 요리명(컬러) + 단계/전체 + 예상 완료. 미니 상태바(다른 요리). 요리 전환 멘트. 딜레이: 재촉 없음, 예상 완료만 조용히 업데이트. 완료 → SCR-013.

### SCR-013: 요리 완료 + 사진 업로드 (등급)
"수고하셨어요! 🎉" + 요리명. 예상 vs 실제 시간 비교. [완성 사진 찍기] (카메라/갤러리) → 등급 +1. 레벨업 시 축하 애니메이션. [사진 없이 완료] → 카운트 안 됨. [홈으로] / [레시피 상세로].

### SCR-014: 쇼츠 변환 — URL 입력 (쇼츠 탭)
"쇼츠 영상을 레시피로 변환해요". URL 입력 + [붙여넣기] + [변환하기]. URL 검증 (youtube.com/shorts/, youtu.be/). 최근 변환 목록.

### SCR-015: 쇼츠 변환 — 결과
로딩 10~30초. 쇼츠 썸네일+제목+원본링크. 단계별 레시피 + AI 추출 재료. [코칭 모드로 시작] + [즐겨찾기 저장]. 실패 시 안내. 캐싱: 같은 URL 즉시 반환.

### SCR-016: 즐겨찾기 목록
정렬 (최신/이름). 쇼츠 변환 레시피도 포함. 스와이프 삭제. 무료 20개 제한.

### SCR-017: 마이페이지
프로필: 이미지 + 닉네임 + **레벨 뱃지 + 칭호**. 등급 영역: 레벨 + 칭호 + 진행률 바 + 총 완료 수. 메뉴: 내 정보 수정, **코칭 설정**, 검색 기록, 앱 설정, 이용약관, 로그아웃, 회원 탈퇴.

### SCR-018: 코칭 설정
코칭 on/off, 음성 속도, 능동 초과 알림, 대기 완료 알림음.

### SCR-019: 회원 탈퇴
30일 유예 후 삭제. Apple 필수 요구사항.
